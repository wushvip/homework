1.4 BCSEClient客户端

BCSEClient线程安全。
表1. 参数描述
参数 参数类型 说明 是否必填 备注
appId string 应用ID 是
userId string 用户ID 是
url string BCSE平台ip地址 是
port int 请求端口 是
secretAccessKey String 系统分配给用户的唯一私钥 是 鉴权时使用
表2. 指定连接池的最大连接数
参数 参数类型 说明 是否必填 备注
maxConnections int 连接池的最大连接数 是
  • 接口定义： 
                
            void setMaxConnections(int maxConnections)
  • 调用方式：

    client(下面的操作将继续使用如下的client)

                
            BCSEClient client = new BCSEClient(“appId”,"userId","ip", port,”secretAccessKey”);

2.5 热词接口

1. 前一个月热词

1)基本描述
表2.5.1.1. 基本描述
接口名称 前一个月热词接口
接口描述 统计并展示前一个的热词信息
2)请求信息
表2.5.1.2. 请求信息
前提要求 无
请求类型 HTTP
请求地址 /statistics/hotWord/month
请求方式 GET
报文格式
content-type：application/json
accept：application/json
接口参数 ?appId=xxx&userId=xxx &pageIndex=xxx&pageNum=xxx
3)参数说明
表2.5.1.3. 请求参数描述
字段 约束 类型 长度 描述 说明
appId 必填 String 50 应用ID
userId 必填 String 50 用户ID
pageIndex 可选 Int 请求当前页码 动态分页所需的页码数（默认从1开始）
pageNum 可选 Int 展现需要显示条数 动态显示条数（默认10条）,最大200 
          
            URL:http://ip:port/bcse/dataAnalysis/hotWord/month?
            appId=1&userId=chenmin@cmss.chinamobile.com&pageIndex=1&pageNum=10
4)响应消息
表2.5.1.4. 返回参数描述
字段 父节点 约束 类型 长度 描述 说明
hotWordList result 必填 Array 热词结果
totalItems result 必填 Int 返回结果总条数
keywords hotWordList 必填 String 关键词
searchCount hotWordList 必填 Int 搜索次数 
          
            {
                "message": "操作成功",
                "code": "",
                "status": "SUCCESS",
                "result": {
                    "hotWordList": [
                        {
                            "keywords": "三国",
                            "searchCount": 37
                        },
                        {
                            "keywords": "植物",
                            "operTime": "",
                            "searchCount": 36
                        },
                        {
                            "keywords": "萌宠",
                            "searchCount": 36
                        }
                    ],
                    "totalItems": 3
                }
            }

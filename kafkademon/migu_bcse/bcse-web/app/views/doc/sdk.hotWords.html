1.9 热词功能

通过这个接口可以将搜索关键词的日志按照天，周，月统计出来。
表1. 参数描述
参数 参数类型 说明 是否必填 备注
dateType String 请求热词的周期 是 DAY/WEEK/MONTH
pageIndex Int 请求当前页码 否 动态分页所需的页码数（默认从1开始）
pageNum Int 展现需要显示条数 否 动态显示条数（默认10条）,最大200
ct String 门户 否
bookType String 渠道 否
调用方式： 
        BCSEHeatWord heatWord=new BCSEHeatWord(client);
        heatWord.setDateType("DAY");
        heatWord.setPageIndex(1);
        heatWord.setPageNum(10);
        String result= heatWord.heatWords();
或者 
        BCSEHeatWord heatWord=new BCSEHeatWord(client);
        HashMap<String Object> opts=new HashMap<>();
        opts.put("dateType" ,"DAY");
        opts.put("pageIndex" ,1);
        opts.put("pageNum" ,10);
        String result= heatWord.heatWords(opts);

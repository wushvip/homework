表1. 参数描述
参数 参数类型 说明 是否必填 备注
fileName String 是
mode String 0表示普通， 1表示自定义
调用方式： 
        
        String fileName;
        String mode;
        BCSESpellDic dic=new BCSESpellDic(client);
        dic.setFileName(fileName);
        dic.setMode(mode);
        dic.upload();
或者 
        String fileName;
        String mode;
        BCSESpellDic dic=new BCSESpellDic(client);
        HashMap<String, Object> opts=new HashMap<>();
        opts.put("fileName" ,fileName);
        opts.put("mode" ,mode);
        String result= dic.upload(opts);

3.4 索引处理

索引处理包括索引更新、索引删除和索引清空。其中索引更新支持本地csv文件上传后更新全量索引。索引删除支持本地csv文件上传后，删除上传部分的索引。索引清空则为清除全部索引。如下图所示：

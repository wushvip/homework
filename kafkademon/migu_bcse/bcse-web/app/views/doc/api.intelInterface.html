2.API接口

2.1 智能联想接口

1. 基本描述

表2.1.1.1. 基本描述
接口名称 智能联想接口
接口描述 根据用户的输入词，自动提示与输入词相关的数据

2. 请求信息

表2.1.2.1. 请求信息
前提要求 无
请求类型 HTTP
请求地址 /suggestion
请求方式 POST
报文格式
content-type：application/json
accept：application/json
接口参数 ?userId=xxx&sortConfig=xxx &appId=xxx&searchQuery=xxx& filterJson={}

3. 参数说明

表2.1.3.1. 请求参数描述
字段 约束 类型 长度 描述 说明
appId 必填 String 50 应用ID
userId 必填 String 50 用户ID
searchQuery 必填 String 100 搜索文本 最大支持100个字符，当搜索文本长度大于100时，自动截取前100个字符
sortConfig 可选 String 50 智能提示配置 规则名称
filterJson 可选 Json 过滤条件筛选 参照搜索接口的filterJson字段规则描述 
        
          URL:http://ip:port/bcse/suggestion?
          userId=chenmin@mcss.chinamobile.com&sortConfig=xxx &appId=1& searchQuery=a&filterJson={}

4. 响应消息

表2.1.4.1. 返回参数描述
字段 父节点 约束 类型 长度 描述 说明
suggestions result 必填 Array 智能提示数据
costTime result 必填 int 耗费时间 
        
          {
              "message": "操作成功",
              "code": "",
              "status": "SUCCESS",
              "result": {
                  "costTime": 1272,
                  "suggestions": []
              }
          }
